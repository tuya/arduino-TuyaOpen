# 日志输出示例

## 功能描述

本示例演示了在涂鸦开发板上使用日志（Log）库的基本用法。展示了如何初始化日志系统并向串口监视器输出调试信息。

## 硬件要求

- 任意涂鸦支持的开发板（T2、T3、T5、ESP32、LN882H 或 XH_WB5E）
- USB 数据线用于串口连接

## 使用说明

1. 在 Arduino IDE 中打开本示例
2. 从工具 > 开发板菜单选择您的涂鸦开发板
3. 上传程序到开发板
4. 打开串口监视器，波特率设置为 115200
5. 观察每秒打印一次的 "Hello" 消息

## 代码功能要点

- **日志初始化**：使用 1024 字节缓冲区初始化日志系统
- **调试输出**：使用 `PR_DEBUG` 宏输出调试级别的消息
- **简单日志记录**：演示将日志记录集成到项目中的最简单方法

## 代码说明

- `Log.begin(1024)`：使用 1024 字节缓冲区初始化日志系统
- `PR_DEBUG("Hello")`：输出一条包含文本 "Hello" 的调试级别日志消息
- 消息在主循环中每秒打印一次

## 输出结果

串口监视器将持续显示：
```
Hello
Hello
Hello
...
```

## 注意事项

本示例使用调试日志级别。Log 库通常支持多个日志级别（ERROR、WARNING、INFO、DEBUG 等）用于不同类型的消息。

# DisplayFill - Fill Screen with Random Colors

## Overview
This example initializes the display and fills the whole screen with a random RGB888 color every 2 seconds.

## Hardware Requirements
- TUYA_T5AI board with a supported display

## How to Use
1. Upload the sketch.
2. Open Serial Monitor at 115200.
3. The screen will change color every 2 seconds.

## Notes
- Random color comes from `random(0xFFFFFF)`.

# LittleFS 演示

## 功能描述

本示例全面展示了在涂鸦开发板上使用 LittleFS（Little File System）库的功能。通过三个测试用例演示了文件操作、目录管理和文件系统操作功能。

## 硬件要求

- 涂鸦支持的开发板（TUYA_T5AI）
- USB 数据线用于串口连接

## 使用说明

1. 在 Arduino IDE 中打开本示例
2. 从工具 > 开发板菜单选择您的涂鸦开发板
3. 上传程序到开发板
4. 打开串口监视器，波特率设置为 115200
5. 观察测试结果，显示文件操作、目录操作和文件复制的执行情况

## 代码功能要点

### 测试1：基本文件操作
- **文件创建与写入**：创建文本文件并写入内容
- **文件读取**：以多种方式读取文件数据（按大小读取、逐字符读取、读取到换行符）
- **文件大小**：获取文件大小信息
- **文件定位**：演示定位到文件中的特定位置
- **位置跟踪**：显示文件中当前读写位置

### 测试2：目录操作
- **创建目录**：创建嵌套目录（`mkdir`）
- **目录存在性检查**：验证目录是否存在
- **目录列表**：打开并读取目录内容
- **目录重命名**：重命名目录
- **删除目录**：删除目录及其子目录

### 测试3：文件复制
- **多文件处理**：同时打开多个文件
- **文件读写**：从一个文件读取并写入另一个文件
- **EOF 检测**：读取时检测文件结束
- **缓冲 I/O**：使用缓冲区实现高效的文件复制

## 代码输出

串口监视器将显示：
- 文件大小和位置信息
- 目录操作结果（成功/失败）
- 各种操作读取的文件内容
- 持续的状态消息显示程序正在运行

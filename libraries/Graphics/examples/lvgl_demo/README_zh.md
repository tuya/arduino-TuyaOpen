# LVGL 演示示例

## 功能描述

此示例演示如何在涂鸦开放平台上使用 LVGL v9.1.0（轻量级通用图形库）。LVGL 是一个流行的开源图形库，用于在嵌入式系统上创建美观的用户界面。

该示例初始化 LVGL，在白色背景的屏幕中央创建一个简单的"Hello World!"标签，并启动 LVGL 任务处理程序。更多关于 LVGL 的信息，请访问 [LVGL 官方网站](https://lvgl.io/)。

## 硬件要求

- 支持显示功能的涂鸦开放平台开发板
- 兼容的 LCD/OLED 显示模块
- 足够的 RAM 用于 LVGL 操作

## 使用说明

1. 根据硬件规格将显示模块连接到开发板。

2. 将程序上传到开发板。

3. 打开串口监视器（波特率 115200）查看初始化日志。

4. 显示屏将在白色背景上居中显示"Hello World!"文本。

5. LVGL 任务持续运行以处理图形更新和动画。

## 功能要点

- **LVGL 集成**：与 LVGL 图形库完全集成
- **硬件注册**：使用板级 API 初始化显示硬件
- **显示锁定**：通过锁定/解锁实现线程安全的显示操作
- **自定义样式**：演示自定义背景和文本颜色
- **任务管理**：使用可配置的周期（5ms）和堆栈大小（8KB）启动 LVGL 任务
- **标签创建**：展示基本的 LVGL 对象创建和对齐

## 代码要点

- 使用 `board_register_hardware()` 注册硬件
- 使用显示名称初始化 LVGL 供应商层：`lv_vendor_init()`
- 使用显示锁定机制确保线程安全：`lv_vendor_disp_lock()` / `lv_vendor_disp_unlock()`
- 将活动屏幕背景设置为白色：`lv_obj_set_style_bg_color(lv_screen_active(), lv_color_white(), LV_PART_MAIN)`
- 创建并居中显示带有黑色文本的标签
- 启动 LVGL 任务处理程序：`lv_vendor_start(5, 1024*8)`，周期 5ms，堆栈 8KB
- 最小化的 loop() 实现，因为 LVGL 在其自己的任务中处理更新

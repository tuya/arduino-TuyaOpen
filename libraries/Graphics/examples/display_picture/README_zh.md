# 显示图片示例

## 功能描述

此示例演示如何使用图形显示库在屏幕上显示图像。它展示了如何处理图像数据、管理显示旋转，以及高效地将图像渲染到显示屏。

该示例支持旋转和非旋转的图像显示，根据显示屏的旋转设置自动选择适当的方法。

## 硬件要求

- 支持显示功能的涂鸦开放平台开发板
- 兼容的 LCD/OLED 显示模块

## 使用说明

1. 将显示模块连接到开发板。

2. 确保您有图像数据文件（通常作为外部数据数组包含）。

3. 将程序上传到开发板。

4. 打开串口监视器（波特率 115200）查看系统信息和调试消息。

5. 显示屏将显示图像，并每 5 秒刷新一次。

## 功能要点

- **图像显示**：将预加载的图像数据渲染到屏幕
- **旋转支持**：自动处理不同的显示旋转角度（0°、90°、180°、270°）
- **系统信息**：显示项目详情和平台信息
- **亮度控制**：将显示亮度设置为 100%
- **高效渲染**：对旋转和非旋转图像使用优化方法
- **帧缓冲管理**：演示正确的图像缓冲区创建和清理

## 代码要点

- 使用外部图像数据数组（`imga_data`、`imga_width`、`imga_height`）
- 检查显示旋转并选择适当的渲染方法：
  - 对于旋转显示：使用 `drawImageRotated()` 函数
  - 对于非旋转显示：使用 `createImageBuffer()` 创建帧缓冲区并直接刷新
- 通过缓冲区清理实现适当的内存管理
- 显示详细的系统信息，包括图像尺寸
- 每 5 秒更新一次显示
- 包含所有显示操作的全面错误处理

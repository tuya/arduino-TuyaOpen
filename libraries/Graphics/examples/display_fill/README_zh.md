# 显示屏填充示例

## 功能描述

此示例演示基本的显示功能，通过使用随机颜色填充整个屏幕。它展示了如何初始化图形显示库、清除屏幕，并以随机生成的颜色持续更新屏幕。

该示例对于测试显示硬件和理解基本显示操作非常有用。

## 硬件要求

- 支持显示功能的涂鸦开放平台开发板
- 兼容的 LCD/OLED 显示模块

## 使用说明

1. 根据硬件规格将显示模块连接到开发板。

2. 将程序上传到开发板。

3. 打开串口监视器（波特率 115200）查看系统信息和调试消息。

4. 观察显示屏，它每 2 秒循环显示一次随机颜色。

## 功能要点

- **显示初始化**：正确初始化显示硬件
- **系统信息**：将项目详情、版本和平台信息打印到串口
- **屏幕清除**：演示如何使用特定颜色清除屏幕
- **随机颜色填充**：使用随机颜色填充整个屏幕
- **显示刷新**：展示正确的缓冲区刷新以更新显示
- **错误处理**：包含显示操作的错误检查

## 代码要点

- 使用 Graphics 库中的 `Display` 类
- 显示系统信息，包括项目名称、版本、编译时间和平台详情
- 获取并显示屏幕分辨率和像素格式信息
- 使用 `tal_system_get_random()` 生成随机颜色
- 使用 `OPERATE_RET` 返回代码实现适当的错误处理
- 每 2 秒用新的随机颜色更新显示
